# Images Requises

Pour que le jeu fonctionne avec les images personnalisées, placez les fichiers suivants dans le dossier `public/` :

1. **asterix_character.png** - Image d'Astérix qui sera affichée sur le crochet
   - Taille recommandée : 40x40 pixels ou plus
   - Format : PNG avec transparence de préférence

2. **gold_nugget.png** - Image de pépite d'or pour remplacer les bulles
   - Taille recommandée : 50x50 pixels ou plus
   - Format : PNG avec transparence de préférence

## Note

Si les images ne sont pas trouvées, le jeu utilisera des placeholders colorés automatiquement.
